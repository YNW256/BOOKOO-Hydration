💧 BOOKOO Hydration
BOOKOO Hydration 是一款高颜值的智能饮水监测系统。它通过蓝牙连接你的电子秤，自动识别你的喝水行为，记录每日摄入量，并生成健康趋势图表。
无需手动记录，拿起杯子喝水，放回即自动记账。
✨ 核心功能
 * 智能感应：自动区分“喝水”、“注水”和“误触”。
 * 幽灵模式 (Ghost UI)：拿起杯子时数值保持不变（变灰），放回后自动结算差值，体验丝滑。
 * 数据可视化：
   * 今日看板：实时显示毫升数 (mL) 和目标进度条。
   * 本周趋势：自动统计过去 7 天的饮水达成情况。
   * 历史时间轴：详细记录每一次饮水的时间和水量。
 * 数据持久化：记录保存在本地，刷新网页或重启电脑数据不丢失。
 * 多语言支持：一键切换 中文 / English。
 * 跨平台适配：完美支持电脑大屏和手机移动端（支持添加到主屏幕）。
🚀 快速开始
1. 环境准备
确保电脑已安装 Python 3.7+。打开终端安装依赖：
pip install fastapi uvicorn bleak

2. 启动服务
在项目目录下运行后端桥接程序：
python server_bridge.py

当看到终端显示 Uvicorn running on http://0.0.0.0:8000 且提示蓝牙已连接，说明服务已就绪。
3. 打开应用
 * 电脑端：直接双击打开 bookoo_hydration.html 文件。
 * 手机端（推荐）：
   * 确保手机和电脑在同一 Wi-Fi 下。
   * 在项目目录新开一个终端运行：python -m http.server 8080
   * 手机浏览器访问：http://<电脑IP地址>:8080/bookoo_hydration.html
⚙️ 使用指南 (首次校准)
为了保证数据准确，初次使用请按以下步骤操作（点击右上角 ⚙️ 进入设置）：
 * 硬件去皮 (HW Tare)：移走秤上所有物品，点击按钮，确保状态栏显示 0g。
 * 设定空杯 (Set Empty)：放上你的空杯子，点击 ∅ Empty。
 * 设定满杯 (Set Full)：把杯子加满水放上去，点击 💧 Full（用于计算杯内水位百分比）。
 * 设定目标：输入你每天计划喝水的总量（例如 2000mL）。
完成！现在你只需要像往常一样喝水，剩下的交给 BOOKOO。
🛠️ 常见操作
 * 暂停记录：在设置中点击 ⏸ Pause，此时你可以把秤拿去冲咖啡或称重，不会影响饮水记录。
 * 撤销误判：如果系统记录错误，点击设置里的 ↺ Undo Last 即可删除上一条记录并回退进度。
 * 清空数据：点击 Clear All 将重置所有历史记录（慎用）。
📱 手机全屏体验
在 iOS Safari 浏览器中，点击底部 分享按钮 -> 添加到主屏幕，即可获得类似原生 App 的全屏沉浸体验。
